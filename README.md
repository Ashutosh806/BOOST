# BOOST

![Boost-Gameplay](https://github.com/Ashutosh806/BOOST/assets/128505054/141ee84f-4605-4fd1-94b6-67d62359dadc)

Fly a rocket from start(blue pad) to finish(green pad) and avoid obstacles in between,colliding with obstacles will restart the level.
Added sound effects for thrusting,crashing and reaching finish.

Added particle effects when thrusting.
